Replacing Default Splash Screen in Kodular apps | Guide

Maayur March 31, 2021 Guides
Note : In this Guide we will be editing the apk using a tool and change contents/values in the apk. If you don't have knowledge regarding it, then don't do any changes except the one mentioned, it may affect your apk. We wont be responsible for it.

So many of you must be wondering that, this can be simply done by setting a image as splash screen from designer, its right but it doesn't looks professional. You can see below the comparison of both outputs. In this guide we will add a xml file for splash screen which will make a layout with icon at centre and fully responsive unlike the image. So without any further ado lets start.

Step 1 : Download and Install Apk Editor Studio from here.

Step 2 : Open the Kodular Compiled Apk with Apk Editor Studio.

Click on Open Contents.

Step 3 : Open the folder res/drawable and delete the splashicon.png image and paste splashicon.xml file there. Download the file from here.

Now as the file is replaced, lets work on the icon and background colour of splash screen.

Step : 4  Open the folder res/drawable and add a icon you want to display on splash screen. its name should be my_icon.png  Don't use big icon use a icon of dimension 96x96 it looks better or you can use any size according to your need.

Step 5 : Open the folder res/values/ and open colors.xml file in a code editor or notepad. Add this line <color name="my_color">#ffffff</color> at last before </resources> tag. You can replace #ffffff with any hex colour value you want. Now save the file.

Now as the editing work is finished close the folder and com back to Apk Editor Studio and click on Save Apk and select the destination and it will generate your apk there.
